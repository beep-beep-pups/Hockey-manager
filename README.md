🏒 Hockey manager — симулятор управления хоккейной командой
Добро пожаловать в игру, где вы становитесь генеральным менеджером хоккейного клуба. Вам предстоит формировать состав, вести команду через регулярный сезон и плей-офф, заключать трансферы и следить за развитием игроков.

🎮 Основные возможности
Управление составом — просмотр игроков, их статистики, покупка и продажа.

Симуляция матчей — реалистичная модель с учётом силы команд, тактики, домашней арены, овертайма и буллитов.

Регулярный сезон — расписание, турнирная таблица, очки по системе 2-1-0.

Плей-офф — серии до 4 побед, чередование домашних матчей.

Развитие игроков — скиллы пересчитываются по итогам сезона на основе набранной статистики.

Трансферы — покупка и продажа игроков с учётом бюджета команды.

Сохранение и загрузка — данные хранятся в JSON-файлах.

📦 Технологии
Python 3.8+

Библиотеки: json, random, math

Опционально: PySide6 для графического интерфейса

🚀 Установка и запуск
Консольная версия
Клонируйте репозиторий или скачайте исходный код.

Установите Python 3.8 или выше.

Убедитесь, что в папке с игрой есть файл teams.json (он создаётся автоматически при первом запуске скрипта teams.py).

Запустите игру командой:

python main.py

Графическая версия (PySide6)
Установите PySide6:

pip install pyside6

Запустите main_gui.py:

python main_gui.py

🕹️ Управление
Консольная версия
Главное меню:

1 — Показать состав команды.

2 — Сыграть матч.

3 — Показать турнирную таблицу.

4 — Открыть трансферный рынок.

5 — Завершить сезон (обновить скиллы игроков).

6 — Сохранить и выйти.

В процессе игры вы сможете выбирать тактику (агрессивная, нейтральная, оборонительная) и влиять на ход матча.

Графическая версия
Все действия выполняются через вкладки:

Состав — таблица игроков с их статистикой.

Матч — кнопка «Следующий матч», отчёт о матче.

Таблица — турнирная таблица лиги.

Трансферы — покупка/продажа игроков.

Настройки — сохранение и загрузка игры.

🧪 Примеры данных
Файл teams.json содержит список команд с игроками. Каждый игрок имеет имя, позицию, скилл (0–100) и цену (для трансферов). Пример:

json
[
  {
    "name": "Dinamo Moscow",
    "players": [
      {"name": "Nick Gusev", "position": "Forward", "skill": 92, "price": 120000},
      {"name": "Ilya Konovalov", "position": "Goalkeeper", "skill": 90, "price": 95000}
    ]
  }
]
